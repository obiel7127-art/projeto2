README (resumo) — Projeto Semáforo Anime

O projeto usa um index.html com uma imagem e quatro botões: Vermelho, Amarelo, Verde e Automático. 

O arquivo semaforo.js troca a imagem (#img) conforme o botão clicado, alterando o semáforo para a cor escolhida.

No modo Automático, o JavaScript alterna as imagens em sequência (vermelho → amarelo → verde) em intervalos.

O arquivo style.css controla o layout, cores e posicionamento dos botões.

Para usar: coloque todos os arquivos na mesma pasta e abra o index.html no navegador.
